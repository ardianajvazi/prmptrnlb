Grid Lists

Grid lists are an alternative to standard list views.

Usage

A grid list consists of a repeated pattern of cells arrayed in a vertical and horizontal layout. Grid lists are best used on similar data types. They help improve the visual comprehension of the content they contain.

Types

  • Image-only
  • Single-line text (or with icon)
  • Two-line text (or with icon)
