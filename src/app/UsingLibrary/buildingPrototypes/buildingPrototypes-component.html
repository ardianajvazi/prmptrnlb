Building Prototypes

This section will guide you through building an interactive, coded prototype using this sytem.

Setting Up Your Environment

The Pattern Library and the future Premera website are both built on Angular2. It may look a bit intimidating, but in the end, it's very much like HTML and CSS (except the HTML looks a little funny and the CSS is actually SASS). The build process (which compiles your SASS into browser-friendly CSS, among other things) happens automatically as we update and save our files, and the app is run via npm start so we can view it in the browser. Once you run the app, updates will show up automatically. Like magic!

In order to prototype using this sytem, all you have to do is download some files, run a few commands in your terminal, and know where to make your updates.
