How To Make Updates

This library is a living document, and will need to be updated with changes to our design system. To get access to the code and make changes, do the following:

  • Visit Bitbucket and create an account if you don't already have one.
  • Ask Ardian to invite you to the repository (the project containing the files).
  • Create a folder on your Desktop or somewhere on your computer.
  • Open a terminal
  • Navigate to your folder using cd Desktop/library
  • Type git clone URLGOESHERE
  • Type cd patternlibrary
  • Type npm install
  • Type gulp watch
  • Open a new terminal window (Command + T on Mac)
  • Type npm start
  • Open up your browser and go to http://localhost:5000/
