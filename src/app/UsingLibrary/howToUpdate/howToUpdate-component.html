How To Make Updates

This library is a living document, and will need to be updated with changes to our design system. To get access to the code and make changes, do the following:

1. Visit GitHub and create an account if you don't already have one. 2. Ask Court to invite you to the repository (the project containing the files). 3. Create a folder on your Desktop or somewhere on your computer. 4. Open a terminal or command line 5. Navigate to your folder using cd Desktop/library 6. Type git clone githubURL 7. Type cd prmptrnlb 8. Type npm install 9. Type npm start 10. Open up your browser and go to http://localhost:3001/
